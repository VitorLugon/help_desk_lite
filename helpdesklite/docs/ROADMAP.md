# Roadmap — HelpDesk Lite

## Etapa 1 — Setup

- Criar projeto Next.js.
- Configurar Tailwind.
- Configurar Prisma.
- Configurar PostgreSQL.
- Configurar Vitest.
- Criar layout base.
- Criar documentação inicial.

## Etapa 2 — Banco

- Criar schema Prisma.
- Criar enums.
- Criar seed com usuários de teste.
- Criar migration inicial.

## Etapa 3 — Autenticação

- Implementar login.
- Implementar logout.
- Proteger rotas internas.
- Criar função para obter usuário atual.

## Etapa 4 — Usuários

- Admin lista usuários.
- Admin cria usuários.
- Admin edita usuários.
- Admin inativa usuários.

## Etapa 5 — Chamados

- Solicitante abre chamado.
- Solicitante lista seus chamados.
- Técnico lista chamados atribuídos.
- Admin lista todos os chamados.
- Admin atribui chamados.

## Etapa 6 — Comentários e status

- Usuários autorizados comentam.
- Técnico altera status.
- Admin altera qualquer status.
- Histórico de status é registrado.

## Etapa 7 — Dashboard

- Cards de indicadores.
- Gráfico por status.
- Gráfico por prioridade.
- Lista de chamados recentes.

## Etapa 8 — Polimento

- Responsividade.
- Estados vazios.
- Loading.
- Erros amigáveis.
- README completo.
- Deploy.
