# Project Brief - HelpDesk Lite

## Resumo

HelpDesk Lite é um sistema de chamados de suporte técnico para pequenas empresas, criado como projeto de portfólio fullstack júnior.

## Problema

Pequenas equipes de suporte precisam registrar pedidos, acompanhar responsáveis, controlar status e visualizar indicadores básicos sem depender de planilhas.

## Solução

A aplicação permite que solicitantes abram chamados, técnicos acompanhem atendimentos atribuídos e administradores gerenciem usuários, atribuições e métricas operacionais.

## Público-alvo

- Recrutadores e entrevistadores técnicos avaliando domínio fullstack.
- Pequenas empresas que precisam de um fluxo simples de suporte.
- Desenvolvedores júnior estudando arquitetura de uma aplicação web completa.

## Diferenciais técnicos

- Next.js com App Router.
- TypeScript em toda a aplicação.
- Modelagem relacional com Prisma e PostgreSQL.
- Validação com Zod.
- Formulários com React Hook Form.
- Testes com Vitest.
- Documentação voltada para leitura de portfólio.
